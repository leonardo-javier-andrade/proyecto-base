# Proyecto 
## Maquetacion de proyecto Web

Se utilizo lo aprendido de HTML y CSS para que el proyecto se viera bastante bien. Se puso mucho foco en los pequeños detalles:

Estructura Completa: Tiene todas las partes que ves en WhatsApp: la lista de chats a la izquierda, la ventana de conversación y la barra para escribir mensajes.

Diseño a Mi Gusto: Le agregué mi toque personal a algunos elementos, como:

Iconos: Usé íconos que me parecieron más bonitos y claros.

Animación al Enviar: Hay un pequeño efecto visual cuando pasas el ratón o haces clic en el botón de "Enviar".

Animación al Adjuntar: El botón para adjuntar archivos también tiene una animación sutil.

### Características 
**Maquetación Fiel:** Estructura completa de la interfaz (lista de chats, cabeceras, área de conversación y pie de entrada).

**Diseño Responsivo Básico:** Se adapta a diferentes tamaños de pantalla (en la medida de lo posible para una réplica web).

**Iconografía Personalizada:** Uso de íconos para mejorar la estética y la claridad de la interfaz. 
Fuente utilizada: "http://yesicon.com"

### Interacciones de Diseño Propias:

**Animación en Botón "Enviar":** Pequeña animación o cambio de estilo al interactuar con el botón de envío de mensajes.

**Animación en Botón "Adjuntar":** Efecto visual sutil al hacer hover o clic en el botón para adjuntar archivos.
